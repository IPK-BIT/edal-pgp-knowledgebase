In addition to the e!Dal-Web submission we provide Desktop applications for data submission.

## IPK data submission

Submit research data for IPK staff - authentication using IPK authentication.

<Tabs syncKey="os">

  <TabItem label="Windows" icon='seti:windows'>
    <Steps>
    
      1. Please download and launch the [windows installer](https://edal-pgp.ipk-gatersleben.de/java/eDAL-Client_IPK-1.0.exe).
      2. Start the app to begin your data upload.
    
    </Steps>
  </TabItem>

  <TabItem label="macOS" icon='apple'>
    <Steps>
    
      1. Please download and launch the [macOS installer](https://edal-pgp.ipk-gatersleben.de/java/eDAL-Client_IPK-1.0.dmg).
      2. Start the app to begin your data upload.
    
    </Steps>
  </TabItem>

</Tabs>
